# Учебный Бот  Нейро-бот

## Описание

Учебный бот предназначен для помощи в обучении и решении задач. Он может работать в нескольких режимах:

*   Чат с ИИ: бот может отвечать на вопросы и предоставлять информацию на различные темы.
*   Текст в голос: бот может синтезировать речь из текста.
*   Анализ фотографий: бот может анализировать фотографии и предоставлять информацию о содержимом.
*   Работа с файлами: бот может анализировать и конспектировать текстовые документы.

## Установка

1.  Установите Python и необходимые библиотеки.
2.  Клонируйте репозиторий и перейдите в папку с проектом.
3.  Создайте файл `.env` и добавьте токен Telegram-бота и другие необходимые настройки.
4.  Запустите бота с помощью команды `python bot.py`.

## Использование

1.  Начните чат с ботом, отправив команду `/start`.
2.  Выберите режим работы, используя кнопки на клавиатуре.
3.  Следуйте инструкциям и отправляйте необходимые данные (текст, фотографии, файлы).


## Требования

*   Python 3.8+
*   Telegram-бот токен
*   Yandex Cloud идентификатор каталога
*   Yandex Cloud API ключ
