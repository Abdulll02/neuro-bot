# Учебный Нейро-бот

## Описание

Учебный бот предназначен для помощи в обучении и решении задач. Он может работать в нескольких режимах:

*   **Чат с ИИ**: бот может отвечать на вопросы и предоставлять информацию на различные темы.
*   **Текст в голос**: бот может синтезировать речь из текста с различными голосами и настройками.
*   **Анализ фотографий**: бот может анализировать фотографии, извлекать текст и решать математические задачи.
*   **Работа с файлами**: бот может анализировать и конспектировать текстовые документы (PDF, DOCX, TXT).

## Архитектура

Проект состоит из следующих компонентов:

### Основные файлы
- `bot.py` - основной файл бота с обработчиками команд и состояний
- `config.py` - конфигурация с настройками API и голосов
- `keyboards.py` - определения inline-клавиатур для Telegram
- `states.py` - состояния для ConversationHandler
- `utils.py` - вспомогательные функции

### Сервисы (`services/`)
- `yandex_ai.py` - интеграция с Yandex GPT для чата, анализа изображений и документов
- `yandex_tts.py` - интеграция с Yandex SpeechKit для синтеза речи

### Зависимости
- `python-telegram-bot[job-queue]==20.7` - фреймворк для Telegram ботов
- `python-dotenv==1.0.0` - загрузка переменных окружения из .env файла
- `requests==2.31.0` - HTTP запросы к API
- `Pillow==10.2.0` - обработка изображений
- `aiofiles==23.2.1` - асинхронная работа с файлами
- `PyPDF2==3.0.0` - парсинг PDF файлов
- `python-docx==0.8.11` - парсинг DOCX файлов

## Установка

### Предварительные требования
- Python 3.8+
- Аккаунт в Yandex Cloud с активированными сервисами:
  - Yandex SpeechKit (для TTS)
  - Yandex GPT (для чата и анализа)
  - Yandex Vision (для OCR изображений)

### Шаги установки
1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd neuro-bot
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   # или
   venv\Scripts\activate     # Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r bot/requirements.txt
   ```

4. Создайте файл `.env` в папке `bot/` со следующими переменными:
   ```
   TELEGRAM_TOKEN=ваш_токен_телеграм_бота
   YANDEX_API_KEY=ваш_yandex_api_key
   YANDEX_FOLDER_ID=ваш_yandex_folder_id
   ```

5. Запустите бота:
   ```bash
   cd bot
   python bot.py
   ```

## Настройка Yandex Cloud

### 1. Создание аккаунта и каталога
1. Зарегистрируйтесь в [Yandex Cloud](https://cloud.yandex.ru/)
2. Создайте каталог в консоли управления
3. Скопируйте ID каталога (поле "Идентификатор каталога")

### 2. Получение API ключа
1. В консоли управления перейдите в раздел "Сервисные аккаунты"
2. Создайте сервисный аккаунт или используйте существующий
3. Перейдите в раздел "Ключи API" → "Создать ключ API"
4. Скопируйте сгенерированный API ключ

### 3. Активация сервисов
Убедитесь, что в вашем каталоге активированы следующие сервисы:
- Yandex SpeechKit
- Yandex GPT
- Vision API

## Использование

### Начало работы
1. Начните чат с ботом, отправив команду `/start`
2. Выберите режим работы, используя кнопки на клавиатуре
3. Следуйте инструкциям и отправляйте необходимые данные

### Режимы работы

#### 💬 Чат с ИИ
- Отправьте текстовое сообщение
- Бот ответит с использованием Yandex GPT
- Поддерживается история разговора (до 10 последних сообщений)

#### 🎤 Текст в голос
- Выберите режим "Текст в голос"
- Отправьте текст для озвучки
- Настройте голос и скорость в разделе "Настройки озвучки"

**Доступные голоса:**
- Алёна (женский, добрый)
- Филипп (мужской, добрый)
- Джейн (женский, добрый/злой)
- Омаж (женский, злой/нейтральный)
- Захар (мужской, добрый/злой/нейтральный)

**Скорости речи:**
- Самый медленный (0.1x)
- Очень медленно (0.5x)
- Медленно (0.8x)
- Нормально (1.0x)
- Быстро (1.3x)
- Очень быстро (1.8x)

#### 📷 Анализ фото
- Отправьте фотографию
- Добавьте подпись для уточнения запроса (опционально)
- Бот извлечет текст с помощью Yandex Vision OCR
- Проанализирует содержимое с помощью Yandex GPT

**Поддерживаемые задачи:**
- Распознавание рукописного текста
- Решение математических примеров
- Анализ учебных материалов

#### 📄 Работа с файлами
- Отправьте файл в формате PDF, DOCX или TXT
- Добавьте подпись с запросом для анализа (опционально)
- Бот извлечет текст и проанализирует его

### Команды
- `/start` - запуск бота и инициализация
- `/help` - справка по использованию

## API и интеграции

### Yandex SpeechKit
Используется для синтеза речи из текста.
- URL: `https://tts.api.cloud.yandex.net/speech/v1/tts:synthesize`
- Формат аудио: MP3 (конвертируется в OGG/OPUS для Telegram)
- Поддержка эмоций и скорости речи

### Yandex GPT
Используется для:
- Чата с ИИ
- Анализа текста с изображений
- Анализа документов

Модели:
- `yandexgpt` / `yandexgpt-lite` - для текстового чата
- `yandexgpt-standard` / `yandexgpt-pro` - для мультимодального анализа

### Yandex Vision
Используется для OCR (распознавания текста на изображениях).
- Извлекает текст перед анализом через GPT

## Troubleshooting

### Ошибки запуска
**"Telegram token is missing"**
- Проверьте наличие переменной `TELEGRAM_TOKEN` в `.env` файле
- Убедитесь, что токен получен от @BotFather в Telegram

**"Yandex API keys are missing"**
- Проверьте `YANDEX_API_KEY` и `YANDEX_FOLDER_ID` в `.env`
- Убедитесь, что API ключ активен в Yandex Cloud

### Ошибки во время работы
**"Ошибка синтеза речи"**
- Проверьте баланс Yandex Cloud
- Убедитесь, что SpeechKit активирован

**"Ошибка анализа фото/файла"**
- Проверьте активацию Vision API и GPT в Yandex Cloud
- Убедитесь в корректности API ключа

**"ffmpeg not found"**
- Установите ffmpeg для конвертации аудио
- Или бот будет отправлять MP3 вместо OGG

### Ограничения
- Максимальный размер файла: ограничен Telegram API
- Длина сообщений: до 4096 символов (автоматическое разбиение)
- История чата: последние 10 сообщений
- Таймаут запросов: 10 секунд

## Безопасность

- Валидация входящих данных
- Обработка ошибок API без раскрытия чувствительной информации
